# 🌸 Iris Classification Web App using Streamlit

This project demonstrates **end-to-end machine learning model deployment** using the Iris dataset.

Users can input flower features and get predictions using:

* Support Vector Machine (SVM)
* Random Forest
* Decision Tree

---

## 🚀 Features

* Multiple ML models
* Interactive UI with Streamlit
* Real-time prediction
* Clean project structure

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Streamlit
* NumPy, Pandas

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python train_model.py
streamlit run app.py
```
